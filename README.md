#  MyRecipeHub  
Web platform for sharing and managing cooking recipes (Spring Boot + Spring Security + Docker)

## Project Description
MyRecipeHub is a web application that allows users to create an account, publish their own recipes, browse recipes shared by others, and save their favorite ones.  
The platform includes secure authentication, a responsive interface, a personal user space, and an administrator role for content moderation.

This project is developed as part of the **Web Programming** course, following the required specifications:
- Authentication using Spring Security (encrypted passwords)
- Role management (USER / ADMIN)
- Image upload for recipes
- Backend built with Spring Boot
- Responsive frontend using Bootstrap
- Deployment with Docker
- Implementation of user stories validated by the professor

---

##  Main Features
- Secure account creation and login
- User profile page
- Add, edit, and delete personal recipes
- Image upload for recipe illustrations
- Browse recipes published by other users
- Add recipes to favorites
- Administrator moderation (delete recipes / manage users)

---

##  User Stories

### **Story 1 — Account Creation**
As a user, I want to create an account so that I can publish my own recipes and access my personal space.

### **Story 2 — Recipe Publishing**
As a logged‑in user, I want to add a recipe with an image so that I can share it with the community.

### **Story 3 — Favorites**
As a user, I want to add recipes to my favorites so that I can easily find them later.

### **Story 4 — Moderation (Admin)**
As an administrator, I want to delete inappropriate recipes so that I can maintain the quality of the platform’s content.
